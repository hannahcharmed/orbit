API overview

The Orbit API is a RESTful HTTP API providing programmatic access to all platform data and intelligence features. Base URL: https://api.orbitapp.io/v1

Creator: 1,000 req/hr

Section 01
Conventions

ID format

Prefixed IDs: prefix_xxxxxxxx. Prefixes: usr users · crt creators · cnt content · ev escape velocity · sig signals · ins insights · agc agencies

Timestamps

All stored as TIMESTAMPTZ UTC. Returned ISO 8601: 2026-03-18T10:00:00Z

Rates

All rates stored as decimals 0–1. Displayed as percentages in UI. Example: 6.8% stored as 0.068.

Badge legend

PK Primary key   FK Foreign key   IDX Indexed   NN Not null

Database

PostgreSQL 15 with TimescaleDB extension for time-series tables. All tables use row-level security (RLS) enforcing creator data isolation.

Section 10
Metric definitions
engagement_rate
engagement_rate = (likes + comments + saves + shares) / reach
Normalized against reach (unique accounts) not followers. Platform-agnostic. Where saves unavailable the formula excludes saves.
gravity_score
score = round( (return_viewer_rate_score × 0.30) + (save_share_depth_score × 0.25) + (comment_quality_score × 0.20) + (cross_platform_score × 0.15) + (follower_velocity_score × 0.10) )
Composite 0–100. Each component normalized to 0–100 before weighting. Recalculated nightly.
baseline_multiple
baseline_multiple = post_engagement_rate / creator_30d_median_engagement_rate
How much a post outperforms the creator's own 30-day median. Value of 2.5 = 2.5× typical performance. Escape velocity threshold is 2.5.
follower_velocity
follower_velocity = (followers_now - followers_28d_ago) / followers_28d_ago
28-day growth rate as decimal. Measures acceleration, not absolute size.
momentum_score (Dark Matter)
momentum_score = normalize(usage_growth_7d×0.50 + niche_relevance×0.30 + acceleration×0.20)
Trend signal strength 0–1. Usage growth is primary driver. Acceleration is second derivative of growth (is it speeding up?).
